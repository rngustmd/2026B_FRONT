- HTML : 하이퍼텍스트 마크업 언어
- 마크업 : <마크업> 텍스트 </마크업>
- 마크업 속성 : <마크업 속성명="속성값"> </마크업>
    0. <html> HTML전체
    0. <head> HTML문서설정 , <title> 문서제목
    0. <body> HTML본문
    1. <h1> ~ <h6>                  : 제목  
    2. <p>                          : 문단 
    3. <br />                       : 줄바꿈
    4. <hr />                       : 가로선
    5. <span>                       : 인라인 구역
    6. <div>                        : 블럭 구역
    7. <ol> , <ul>                  : 목록
    * <li>                          : ol 또는 ul 마크업 안에 항목
    8. <a href="링크주소/파일경로">  : 하이퍼링크
    9. <img src="파일경로" />        : 이미지
    10. <audio src="파일경로"> , <video src="파일경로"> : 오디오/비디오
    * controls속성 : 컨트롤러 제공
    * autoplay속성 : 자동실행(브라우저정책상 차단)
    * muted속성 : 음소거 제공
    11. <iframe> : 외부 컨텐츠를 포함 (퍼가기)
